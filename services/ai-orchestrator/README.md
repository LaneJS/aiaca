# AI Orchestrator Service (Node/TypeScript)

Placeholder service for routing scan outputs through AI models.

## Getting Started
- Install dependencies: `npm install`
- Run locally: `npx nx serve ai-orchestrator`
- Run tests: `npx nx test ai-orchestrator`
- Build production bundle: `npx nx build ai-orchestrator`
